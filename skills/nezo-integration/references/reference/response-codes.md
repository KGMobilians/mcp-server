# 공통 응답 코드

내죠여왕 API의 공통 응답 코드(result_cd) 목록입니다.

## 응답 코드 체계

모든 API 응답에는 `result_cd`(결과 코드)와 `result_msg`(결과 메시지)가 포함됩니다.

## 성공 코드

| result_cd | result_msg |
|-----------|------|
| `0000` | 요청이 정상적으로 처리됨 |

## 오류 코드

| result_cd | result_msg |
|-----------|-----------|
| `0001` | 필수 파라미터가 누락되었습니다. {내용} |
| `0002` | 처리 중 오류가 발생하였습니다. | 
| `0003` | 월 이용 한도 초과 | 
| `0004` | 블랙리스트에 등록된 발신자 입니다. | 
| `0005` | 유효한 서비스 아이디가 아닙니다. | 
| `0006` | 이미 요청된 가맹점 거래번호 입니다. | 
| `0007` | MAC이 유효하지 않습니다. | 
| `0008` | 파라미터 값이 유효하지 않습니다.(복호화 실패) | 
| `0009` | 거래내역 저장 실패 | 
| `0010` | 가맹점 거래번호 값이 유효하지 않습니다. | 
| `0011` | 걸재정보가 없습니다. | 
| `0012` | 결제 취소 실패 | 
| `0013` | 결제 취소 에러 | 
| `0014` | 이미 등록되었습니다. | 
| `0015` | 이미 취소 되었습니다. | 
| `0016` | 결제 가능 시간이 초과되었습니다. | 
| `0017` | 전문 유효 시간이 초과되었습니다. | 
| `0018` | 블랙리스트에 등록되지 않았습니다. | 
| `0019` | 이미 결제가 완료된 건입니다. | 
| `0021` | 가맹점의 결제 결과 URL(return_url)이 정상적인 URL형태가 아닙니다. | 
| `0022` | 가맹점의 도메인이 정상적이지 않습니다. | 
| `0023` | 이미 자동결제가 등록된 아이디 입니다. | 
| `0024` | 자동 결제 등록 정보가 없습니다. | 
| `0025` | 해당 정보는 자동결제가 해지된 상태입니다. | 
| `0026` | 결제 요청 한도 설정값을 확인하시기 바랍니다. | 
| `0027` | 자동결제를 이용할 수 없습니다. | 
| `A998` | 허용된 IP가 아닙니다. | 
| `A999` | 자동결제가 실패했습니다. | 
| `E999` | SYSTEM ERROR |

> **참고**: 위 응답 코드는 대표적인 코드이며, 실제 운영 환경에서는 추가 코드가 존재할 수 있습니다. 정확한 전체 코드 목록은 KG파이낸셜 기술지원에 문의하세요.
