# 코드표

MOBILPAY API에서 사용하는 결제수단, 신용카드, 간편결제, 금융기관, 이동통신사 코드표입니다.

## 결제수단코드표

거래등록, 결제승인, 결제취소 등 API의 `cash_code` 필드에 사용하는 코드입니다.

| 결제수단명 | 코드 | 설명 |
|-----------|------|------|
| 휴대폰 | `MC` | 통신사 소액결제 |
| 신용카드 | `CN` | 일반 카드결제 및 간편결제 포함 |
| 실계좌이체 | `RA` | 실시간 계좌이체 (뱅크페이) |
| 가상계좌 | `VA` | 가상계좌 발급 후 입금 |

## 간편결제코드표

거래등록 시 `pay_options.ep_code`에 사용하는 간편결제사 코드입니다.

| 간편결제명 | 코드 | 간편결제명 | 코드 |
|-----------|------|-----------|------|
| 카카오페이 | `KKO` | 애플페이 | `APP` |
| 토스 | `TOS` | 페이코 | `PYC` |
| 삼성페이 | `SSP` | 엘페이 | `LTP` |
| 네이버페이 | `NAV` | SSG페이 | `SSG` |

### 간편결제 참고사항

- **네이버페이**: iframe 환경에서 지원 불가. `call_type`을 `P` 또는 `S`로 설정
- **SSG페이**: `ep_no`(구매자 휴대폰번호) 필수
- **네이버페이 포인트 결제**: `ep_issue_type`, `ep_issue_no` 필수 (현금영수증)
- **카카오페이 카드사 고정**: `ep_card_code` 사용 가능 (카카오페이, 네이버페이, 토스 지원)
