# mobilpay_mcp 프로젝트 개발공수 분석 (WBS)

## Context

바이브코딩(AI 활용 개발)으로 구축한 mobilpay_mcp 프로젝트의 개발공수를 정량적으로 산정하여, AI 활용 전/후의 리소스 절감 효과를 측정하기 위한 분석입니다.

## WBS: 중급 개발자 1명 기준 (Man-Day)

> **전제조건**: 중급 개발자 = 경력 3~5년, TypeScript 경험 있음, MCP 프로토콜은 처음 접함
> **1 Man-Day = 8시간 순수 개발시간**
