### 핵심 기술 결정사항
| 항목 | 선택 | 이유 |
|---|---|---|
| 트랜스포트 | STDIO | IDE(Cursor, Kiro 등)가 프로세스 직접 관리 |
| 검색 | BM25 | 외부 의존성 없이 경량 키워드 검색 |
| 문서 파싱 | remark-parse (AST) | heading 기반 정확한 청크 분리 |
| 배포 | NPM scoped package | `npx`로 설치 없이 즉시 실행 가능 |
| 문서 카탈로그 | llms.txt | LLM 친화적 표준 문서 인덱스 |
