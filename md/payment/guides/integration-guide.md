# 적용가이드

MOBILPAY 결제 연동 시 결제수단별 주의사항, 서비스 제공 범위, 결제창 크기 등 연동에 필요한 참고사항을 안내합니다.

## 결제수단

MOBILPAY에서 지원하는 결제수단입니다.

| 결제수단 | cash_code | 설명 |
|----------|-----------|------|
| 휴대폰 | `MC` | 통신사 소액결제 |
| 신용카드 | `CN` | 신용카드 일반 결제 |
| 실계좌이체 | `RA` | 실시간 계좌이체 (뱅크페이) |
| 가상계좌 | `VA` | 가상계좌 발급 후 입금 방식 |
| 간편결제 | `EP` | 카카오페이/토스/네이버페이/삼성페이/페이코/엘페이/SSG페이/애플페이 (`ep_code`로 구분) |
| 교통카드(모바일티머니) | `TM` | 티머니페이. `hybrid_pay=Y` 필수 |

## 연동 참고사항

### 간편결제 네이버페이

네이버페이(`ep_code=NAV`)는 iframe 환경에서 지원되지 않습니다. 거래등록 시 `call_type`을 `P`(popup) 또는 `S`(self)로 설정해야 합니다.

### 하이브리드 + 통합 결제

하이브리드결제(`hybrid_pay=Y`)와 통합결제(`integrate_pay=Y`)를 동시에 사용하면 실계좌이체(`cash_code=RA`)는 결제수단으로 노출되지 않습니다.

### 결제창 호출 방식

결제창 호출 시 form submit 방식은 지양하세요. IE11에서 about:blank 새 창 문제가 발생할 수 있습니다. `window.open()`, `location.href`, iframe `src` 할당 방식을 권장합니다.
